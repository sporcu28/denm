=== Görsel SEO Asistanı ===
Yazar: Senin Adın
Sürüm: 1.0

== Açıklama ==
Bu eklenti, WordPress sitenizdeki eksik alt etiketli görselleri bulur, yapay zekâ ile öneri sunar ve tek tıkla toplu güncelleme imkanı sağlar. Ayrıca CSV/JSON dışa aktarım ve SEO bilgilendirme kutusu içerir.

== Kurulum ==
1. Bu klasörü `wp-content/plugins/` dizinine yükleyin.
2. WordPress admin panelinden eklentiyi etkinleştirin.
3. Sol menüde "Görsel SEO Asistanı" sekmesini göreceksiniz.

== Kullanım ==
- Eksik alt etiketi olan görselleri admin panelinde görebilirsiniz.
- "Tümünü Güncelle" ile önerilen alt metinleri topluca kaydedebilirsiniz.
- CSV veya JSON olarak dışa aktarabilirsiniz.

== OpenAI API Anahtarı Ekleme ==
Yapay zekâ ile alt metin önerisi almak için `wp-config.php` dosyanıza aşağıdaki satırı ekleyin:

    define('GSA_OPENAI_API_KEY', 'OPENAI_API_ANAHTARINIZ');

== Destek ==
Sorularınız için eklenti geliştiricisine ulaşabilirsiniz.
